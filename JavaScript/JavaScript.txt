button onclick="clique()"
button tag
onclick parametro
"clique()" Nome do evento

<button onclick="clique()">Clique em mim</button>
    <script>
        function clique() {
            window.alert('Você clicou')
            
        }
    </script>

O evento onclick executa determinada funcionalidade quando um botão é clicado.

<button onclick="inicio()">Vamos começar?</button>

   <script>
    function inicio() {
        let name = window.prompt('Qual é o seu nome please?')
        window.alert(`Ola ${name} é um prazer te conhecer.`)
        
    }
   </script>
